## 📂 프로젝트명 : SpaceMoon
- 개발 인원 : 5명
- 기여도 : 공지사항, FAQ, 관리자페이지 구현
- 참고 사이트 : space cloud  https://www.spacecloud.kr/

<br>
<br>

##  ⚒ 개발환경

- 프레임워크 : Spring, myBatis
- DB : Oracle
- WAS : Apache Tomcat
- VIEW : JSP MODEL2(EL, JSTL), HTML, CSS, bootstrap, JAVASCRIPT, jQuery
- 협업 툴 : svn, ERD cloud
- 빌드자동화도구 : maven
